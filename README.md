# find-me
## 미아방지 wearable system

> 기능
> 위치 추적 기능     
> 팔찌로 데이터 전송 받는 기능      
> 관제센터의 안내사항을 받는 기능      
> 앱으로 현위치와 지도의 좌표를 확인하는 기능
